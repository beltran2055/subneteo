¿Qué es el subneteo?

El subneteo es el proceso de dividir una red IP en subredes más pequeñas, para mejorar la organización, seguridad y uso eficiente del espacio de direcciones IP.

Pasos para realizar el subneteo

  1. Determinar la cantidad de subredes o hosts requeridos.
  2. Calcular cuántos bits se necesitan para las subredes.
  3. Determinar la nueva máscara de subred.
  4. Calcular los rangos de IP válidas por subred.
  5. Asignar IPs a los dispositivos.

¿A qué subred pertenece una dirección IP?

Para saber a qué subred pertenece una IP, se aplica la máscara de subred mediante una operación AND:

  • IP: 192.168.1.130
  • Máscara: 255.255.255.192 (/26)
  • Resultado: IP pertenece a la subred 192.168.1.128/26

¿Qué dirección IP le corresponde a un número de host?

Para saber qué IP corresponde al host #X dentro de una subred:

  • Subred: 192.168.1.0/28
  • Host #5
  • Rango válido: 192.168.1.1 a 192.168.1.14
  • IP del host #5: 192.168.1.5

Tips de subneteo

  • Siempre calcula los bits necesarios.
  • Recuerda que los rangos empiezan después del ID de subred.
  • La última IP es la de broadcast.
